Экзаменационный проект по веб-технологиям Князевой Т.И., 241-326.


Папка images - ассеты с изображениями для сайта.


В случае, если хостинг не заработает, сайт открывать локально: 
1. Cкачиваем директорию и распаковываем в отдельную папку.
2. Далее используем последовательность команд в командной строке (Command Prompt):

+ cd /путь до папки/

+ python -m http.server port (порт от 80)

3. Подключение к сайту происходит через браузер с помощью localhost:port.

Внешний вид сайта (без рабочего api) можно увидеть здесь: https://daildaul.github.io/exam_web/.
